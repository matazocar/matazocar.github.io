How this works

Every file in the posts/ folder that ends in .md becomes a page under blog/. Each post starts with a small block of metadata:

---
title: My post title
date: 2026-06-14
tags: math, notes
summary: One sentence shown on the blog index.
---

After the --- you can write normally in Markdown: bold, italic, inline code, links, lists, blockquotes, and code blocks all work.

A blockquote looks like this.
  • First point
  • Second point
  • Third point

When you push your changes, GitHub Actions runs node build.js automatically, which regenerates this page and updates the blog index.
